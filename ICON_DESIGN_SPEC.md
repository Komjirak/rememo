# 🎨 앱 아이콘 디자인 스펙

스플래시 화면 중앙 이미지를 기반으로 한 앱 아이콘 디자인 사양입니다.

## 📐 기본 사양

- **크기**: 1024x1024px (필수)
- **형식**: PNG
- **배경**: 검은색 (#000000)

## 🎨 디자인 요소

### 1. 아이콘 컨테이너
- **크기**: 전체 캔버스의 87.5% (896x896px)
- **위치**: 중앙 정렬
- **배경색**: 검은색 (#000000)
- **Border Radius**: 22.5% (iOS 스타일, 약 201.6px)

### 2. 모서리 브래킷
- **색상**: Accent Teal (#2DD4BF)
- **크기**: 각 모서리 32px (1024 기준으로 스케일)
- **두께**: 3px
- **위치**: 4개 모서리 (상단 좌/우, 하단 좌/우)
- **글로우 효과**: 
  - 색상: #2DD4BF (opacity 60%)
  - Blur: 15px

### 3. 중앙 원형 테두리
- **위치**: 중앙
- **반지름**: 아이콘 크기의 35.7% (약 320px)
- **색상**: 흰색 (opacity 10%)
- **두께**: 1px

### 4. 중앙 아이콘 (auto_awesome)
- **위치**: 정중앙
- **크기**: 아이콘 컨테이너의 25% (약 224px)
- **색상**: Accent Teal (#2DD4BF)
- **디자인**: 
  - 중앙에 5개 뾰족한 별 모양
  - 주변에 6개의 작은 스파클 (4개 뾰족한 별)
- **글로우 효과**:
  - 배경 글로우: #2DD4BF (opacity 30%), Blur 8px
  - 아이콘 그림자: #2DD4BF (opacity 80%), Blur 12px

## 🎨 색상 팔레트

```
검은색 배경: #000000
Accent Teal: #2DD4BF
흰색 테두리: #FFFFFF (10% opacity)
```

## 📝 제작 가이드

### Figma/Sketch/Adobe XD 사용 시:

1. **캔버스 생성**: 1024x1024px
2. **배경 레이어**: 검은색 (#000000) 채우기
3. **아이콘 컨테이너**: 
   - 896x896px 사각형
   - Border Radius: 201.6px (22.5%)
   - 검은색 배경
4. **모서리 브래킷**: 
   - 각 모서리에 32px 크기의 L자 모양
   - 색상: #2DD4BF
   - Stroke: 3px
   - Outer Glow: #2DD4BF, 60% opacity, 15px blur
5. **중앙 원형 테두리**: 
   - 640x640px 원 (중앙 정렬)
   - Stroke: 1px, 색상: #FFFFFF, 10% opacity
6. **중앙 아이콘**: 
   - Material Icons의 `auto_awesome` 아이콘 사용
   - 또는 별/스파클 모양 직접 그리기
   - 색상: #2DD4BF
   - 크기: 약 224px
   - Outer Glow 추가

### 온라인 도구 사용 시:

- **Canva**: 1024x1024 템플릿 사용
- **Photopea** (무료 Photoshop 대체): 레이어로 제작
- **GIMP**: 무료 이미지 편집 도구

## 🚀 다음 단계

이미지 파일을 `assets/icon.png`로 저장한 후:

```bash
flutter pub run flutter_launcher_icons
```

이 명령어가 자동으로 모든 플랫폼의 아이콘을 생성합니다!
